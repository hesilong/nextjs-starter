# Install OpenClaw

## Requirements
- Node.js 20.x
- pnpm 10.12.4

## Quick Start
```bash
pnpm install
pnpm dev
```

## Verify
Open `http://localhost:3000` and confirm the homepage loads.

## Common Issues
- If `pnpm` is missing, install it globally first.
- If port 3000 is occupied, use `pnpm dev -- --port 3001`.
