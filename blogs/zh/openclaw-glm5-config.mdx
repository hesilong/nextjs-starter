## 什么是 GLM-5？

GLM-5 是智谱 AI 推出的、面向复杂系统工程与长链路智能体任务的大模型，目标是提升通用智能在复杂场景下的效率与可靠性。相较 GLM-4.5，GLM-5 的规模从 355B 参数（32B 激活）扩展到 744B 参数（40B 激活），预训练数据从 23T 提升到 28.5T tokens，并引入 DeepSeek Sparse Attention（DSA）以在保持长上下文能力的同时显著降低部署成本。

在后训练方面，GLM-5 引入了异步强化学习基础设施 slime，以更高吞吐与更细粒度的迭代提升模型表现。综合预训练与后训练进展，GLM-5 在推理、编程和智能体任务上相较 GLM-4.7 有明显提升，并在多项学术基准上达到开源模型中的领先水平。

## 配置流程（3 步搞定）

### 步骤 1：准备 API 密钥

1. 访问 <a href="https://bigmodel.cn/" target="_blank" rel="nofollow noopener noreferrer" className="text-blue-600 underline font-semibold">智谱 AI 开放平台</a>
2. 注册或登录账户
3. 在控制台创建 API 密钥
4. 复制并保存 API 密钥
![GLM-5 API 密钥创建示意图](https://cdn.openclawai.cv/openclaw-glm5-configure-1.png)

请记录以下信息：

1. API 密钥：`xxxxxxxxx`
2. Base URL：`https://open.bigmodel.cn/api/paas/v4`
3. 模型 ID：`glm-5`

### 步骤 2：使用配置生成器

访问 <a href="https://www.openclawai.cv/tools/openclaw-model-config-generator" target="_blank" rel="nofollow noopener noreferrer" className="text-blue-600 underline font-semibold">OpenClaw 配置生成器</a>，按以下顺序填写。

#### 2.1 添加 Provider

在 "Providers" 部分点击 "Provider 1"，填写：

1. Provider key：`zhipu`
2. Base URL：`https://open.bigmodel.cn/api/paas/v4`
3. API key：你的 GLM-5 API 密钥

#### 2.2 配置模型参数

在 "Model 1" 部分设置：

1. Model ID：`glm-5`
2. Model name：`glm-5-assistant`
3. Max tokens：`4096`
4. Reasoning：开启（推荐）
5. Text input：开启
6. Image input：按需开启

#### 2.3 设置主模型

将 "GLM-5" 设置为 "Primary model"。

### 步骤 3：应用配置

生成器会提供 JSON 配置，点击 "Copy" 复制到剪贴板。
![OpenClaw 配置生成器 JSON 输出示意图](https://cdn.openclawai.cv/openclaw-glm5-configure-2.png)

#### 方案 A：配置文件合并（推荐）

1. 在配置生成器中上传你当前的 `openclaw.json` 配置文件。
2. 让工具自动合并新配置（例如 GLM-5 的 provider 和 model）。
3. 下载合并后的配置文件并覆盖原有配置。

建议操作前先备份：

```bash
cp ~/.openclaw/openclaw.json ~/.openclaw/openclaw.json.backup
```

#### 方案 B：OpenClaw CLI 快速设置

```bash
openclaw config set agents.defaults.model "glm-5"
openclaw config set agents.defaults.baseUrl "https://open.bigmodel.cn/api/paas/v4"
openclaw config set agents.defaults.apiKey "你的GLM-5-API-密钥"
openclaw config set agents.defaults.temperature 0.7
```

## 故障排除

### Invalid API key

原因：API 密钥错误或过期  
解决：重新检查智谱 AI 控制台，确认 API 密钥正确

### Connection timeout

原因：网络连接问题  
解决：

```bash
openclaw config set agents.defaults.timeout 60000
ping open.bigmodel.cn
```

### Model not found

原因：模型 ID 错误  
解决：确认使用 `glm-5`

### Poor response quality

原因：温度参数不当  
解决：

```bash
openclaw config set agents.defaults.temperature 0.5
```
